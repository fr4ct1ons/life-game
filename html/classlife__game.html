Default constructor.

Starts an instance of Conway's Game of Life.

Parameters
filename- Name of the file used for input of the game's starting cells.
folder- Folder used for saving the images.
dead- Color representing the dead cells.
alive- Color representing the alive cells.
outf- File used for storing the game's generation log.
maxGen- Limit of generations the program can have.
img_blockSize- Size of the block used to represent a cell, when rendering the images.
